# TARA Data summary

### Installation

```
git clone https://github.com/cjneely10/TARA-Analysis.git
cd TARA-Analysis
./install.sh
```

### Usage

Run the visualizer with the commands:

```
conda activate TARA-Analysis
./run.py
```
